Overchan

Meta Imageboard Client für Android

Zum Projekt auf GitHub

Overchan ist eine mobile Anwendung für das Surfen Imageboards für Android.

Das Projekt ist eine freie und Open-Source-Software (unter der GNU GPLv3-Lizenz).

Einige Funktionen

  • Individuelle Einstellungen für jede Imageboard:
    Passwort zum Löschen von Posts, HTTPS/SSL, Proxy
  • Automatisches Verstecken durch reguläre Ausdrücke
  • Automatische Aktualisierung aller offener Tabs im Hintergrund
  • Senden von Posts im Hintergrund
  • Bewahrung der Threads mit allen Anhängen in HTML mit Dollchan Script
  • Anzeige der Antworten sowohl in Einzelform als auch in einer Liste
  • Unterstützung von Tablet-Schnittstelle
  • Verschiedene Themen
  • Unterstützung ab Android 1.6
