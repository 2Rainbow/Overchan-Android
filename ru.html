Overchan

Meta Imageboard Client for Android

View project on GitHub

Overchan – мобильный браузер различных имиджбордов для Android.

Проект является свободным ПО с открытым исходным кодом (лицензия GNU GPLv3).

Некоторые возможности

  • Отдельные настройки для каждой имиджборды:
    пароль для удаления постов, HTTPS/SSL, прокси
  • Автоматическое скрытие по регулярным выражениям
  • Автоматическое обновление всех открытых вкладок в фоновом режиме
  • Отправка сообщений в фоновом режиме
  • Сохранение тредов со всеми вложениями в HTML с куклоскриптом
  • Отображение ответов к постам как в виде отдельных ссылок, так и в виде списка
  • Поддержка планшетного интерфейса
  • Различные темы оформления
  • Поддержка начиная с версии Android 1.6
