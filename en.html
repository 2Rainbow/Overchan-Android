Overchan

Meta Imageboard Client for Android

View project on GitHub

Overchan is a mobile application for browsing imageboards for Android

The project is free and open-source software (licensed under the GNU GPLv3).

Some features

  • Separate preferences for each imageboard:
    password for post deletion, HTTP/SSL, proxy
  • Automatic hide posts and threads using regular expressions
  • Automatic update all open tabs in background
  • Sending posts in background
  • Saving threads with all attachments in HTML with Dollchan
  • Two ways to display replies to posts: as separate references and as list
  • Support the tablet interface
  • Themes
  • Support Android 1.6 and newer
