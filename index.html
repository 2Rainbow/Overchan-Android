Overchan - мобильный браузер различных имиджборд для Android

Некоторые возможности

  • Индивидуальные настройки для каждой имиджборды: пароль для удаления постов, HTTPS/SSL, прокси
  • Автоскрытие по регулярным выражениям
  • Автоматическое обновление всех открытых вкладок (в фоновом режиме)
  • Сохранение тредов в HTML с куклоскриптом
  • Отправка сообщений в фоновом режиме
  • Отображение ответов
  • Поддержка планшетного интерфейса
